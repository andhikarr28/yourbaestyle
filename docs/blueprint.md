# **App Name**: YourbAIstyle

## Core Features:

- AI-Powered Chatbot (Indonesia): An AI Chatbot that responds to employee questions in Bahasa Indonesia, referencing only approved information stored in Firestore. The chatbot employs a tool that will ensure that its answers incorporate the correct details about operations, SOPs, product information, and documented tacit knowledge.
- Knowledge Library: Display of organizational knowledge, consisting of SOPs, product knowledge, and tacit knowledge extracted from Firestore.
- Content Management (Admin): Authorized users can create, update, and delete data related to SOPs, product knowledge, and tacit knowledge, which is then stored in Firestore.
- User Authentication and Roles: Role-based access control to ensure only authorized roles (Owner/Admin) can modify data. Utilizes Firebase Authentication to secure the application.
- Chat Log and Interaction Tracking: Logging user interactions with the chatbot, which is stored in Firestore for auditing and improvement purposes.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional, trustworthy feel.
- Background color: Light Indigo (#E8EAF6) provides a clean, subtle backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and key information.
- Body text: 'PT Sans' a modern, warm sans-serif for easy reading.
- Headline text: 'Space Grotesk' a techy sans-serif for headlines and short amounts of body text.
- Use clean and modern icons that visually represent different knowledge categories and actions.
- Clean, card-based layout for the Knowledge Library to organize information clearly. Chatbot interface with a clear distinction between user input and AI responses.
- Subtle animations and transitions to enhance user experience and provide feedback on interactions.
